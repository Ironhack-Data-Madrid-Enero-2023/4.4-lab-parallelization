class="vector-body">\n\t\t<div id="siteSub" class="noprint">Fra Wikipedia, den frie encyklopedi</div>\n\t\t<div id="contentSub"><div id="mw-content-subtitle"></div></div>\n\t\t<div s\xc3\xb8k</a>\n\t\t<div id="mw-content-text" class="mw-body-content mw-content-ltr" lang="nb" dir="ltr"><div class="mw-parser-output"><p><b>Datavitenskap</b> er et <a href="/wiki/Tverrfaglig" class="mw-redirect" title="Tverrfaglig">tverrfaglig</a> felt som bruker <a href="/wiki/Statistikk" title="Statistikk">statistikk</a>, <a href="/wiki/Vitenskapelig_metode" title="Vitenskapelig metode">vitenskapelige metoder</a>, <a href="/wiki/Prosess_(informatikk)" title="Prosess (informatikk)">prosesser</a>, <a href="/wiki/Algoritme" title="Algoritme">algoritmer</a> og systemer for \xc3\xa5 generere <a href="/wiki/Kunnskap" title="Kunnskap">kunnskap</a> og <a href="/wiki/Forretningsinnsikt" title="Forretningsinnsikt">innsikt</a> fra st\xc3\xb8yete, <a href="/wiki/Struktur" title="Struktur">strukturerte</a> og <a href="/w/index.php?title=Ustrukturerte_data&amp;action=edit&amp;redlink=1" class="new" title="Ustrukturerte data (siden finnes ikke)">ustrukturerte data</a>.<sup id="cite_ref-1" class="reference"><a href="#cite_note-1">&#91;1&#93;</a></sup><sup id="cite_ref-2" class="reference"><a href="#cite_note-2">&#91;2&#93;</a></sup> Datavitenskap er relatert til <a href="/wiki/Datautvinning" title="Datautvinning">datautvinning</a>, <a href="/wiki/Maskinl%C3%A6ring" title="Maskinl\xc3\xa6ring">maskinl\xc3\xa6ring</a> og <a href="/wiki/Stordata" title="Stordata">stordata</a>.\n</p><p>Datavitenskap er et har blitt beskrevet som et konsept for \xc3\xa5 forene <a href="/wiki/Statistikk" title="Statistikk">statistikk</a>, dataanalyse, <a href="/wiki/Informatikk" title="Informatikk">informatikk</a> og deres relaterte metoder for \xc3\xa5 forst\xc3\xa5 og <a href="/wiki/Analyse" title="Analyse">analysere</a> faktiske <a href="/wiki/Fenomen" title="Fenomen">fenomen</a> med <a href="/wiki/Data" title="Data">data</a>.<sup id="cite_ref-3" class="reference"><a href="#cite_note-3">&#91;3&#93;</a></sup> Det tas i bruk teknikker og <a href="/wiki/Teori" title="Teori">teori</a> fra felt som <a href="/wiki/Matematikk" title="Matematikk">matematikk</a>, statistikk, <a href="/w/index.php?title=Datateknikk&amp;action=edit&amp;redlink=1" class="new" title="Datateknikk (siden finnes ikke)">datateknikk</a>, informatikk, <a href="/wiki/Informasjonsvitenskap" title="Informasjonsvitenskap">informasjonsvitenskap</a> og <a href="/wiki/Domenekunnskap" class="mw-redirect" title="Domenekunnskap">domenekunnskap</a>.<sup id="cite_ref-4" class="reference"><a href="#cite_note-4">&#91;4&#93;</a></sup> Datavitenskap er imidlertid forskjellig fra datateknikk (<i>computer science</i>) og informasjonsvitenskap. <a href="/wiki/Turing-prisen" title="Turing-prisen">Turing-prisvinneren</a> Jim Gray s\xc3\xa5 for seg datavitenskap som et et fjerde paradigme av vitenskap (<a href="/wiki/Empirisk_forskning" title="Empirisk forskning">empirisk</a>, <a href="/wiki/Grunnforskning" title="Grunnforskning">teoretisk</a>, <a href="/wiki/Beregningsvitenskap" title="Beregningsvitenskap">beregnet</a>, og n\xc3\xa5 datadrevet), og hevdet at alt om vitenskap endrer seg p\xc3\xa5 grunn av innvirkningen fra <a href="/wiki/Informasjons-_og_kommunikasjonsteknologi" title="Informasjons- og kommunikasjonsteknologi">informasjonsteknologi</a> og <a href="/w/index.php?title=Informasjon_eksplosjon&amp;action=edit&amp;redlink=1" class="new" title="Informasjon eksplosjon (siden finnes ikke)">informasjonseksplosjonen</a>.<sup id="cite_ref-TansleyTolle2009_5-0" class="reference"><a href="#cite_note-TansleyTolle2009-5">&#91;5&#93;</a></sup><sup id="cite_ref-BellHey2009_6-0" class="reference"><a href="#cite_note-BellHey2009-6">&#91;6&#93;</a></sup>\n</p><p>En <b>dataviter</b> er en person som lager programmeringskode og kombinerer den med statistisk kunnskap for \xc3\xa5 skape innsikt fra data.<sup id="cite_ref-7" class="reference"><a href="#cite_note-7">&#91;7&#93;</a></sup>\n</p><p>Det er fortsatt ingen konsensus om definisjonen av datavitenskap, og det anses av noen \xc3\xa5 v\xc3\xa6re en <a href="/w/index.php?title=Moteord&amp;action=edit&amp;redlink=1" class="new" title="Moteord (siden finnes ikke)">moteord</a>.<sup id="cite_ref-8" class="reference"><a href="#cite_note-8">&#91;8&#93;</a></sup> <a href="/wiki/Stordata" title="Stordata">Stordata</a> er et relatert markedsf\xc3\xb8ringsbegrep.<sup id="cite_ref-:5_9-0" class="reference"><a href="#cite_note-:5-9">&#91;9&#93;</a></sup> Datavitere er ansvarlige for \xc3\xa5 bryte ned stordata til brukbar informasjon og lage programvare og algoritmer som hjelper selskaper og organisasjoner med \xc3\xa5 styre driften optimalt.<sup id="cite_ref-:6_10-0" class="reference"><a href="#cite_note-:6-10">&#91;10&#93;</a></sup>\n</p>\n<h2><span id="Se_ogs.C3.A5"></span><span class="mw-headline"
